
## Инструкция для администратора: начальная настройка и эксплуатация

*Глобальные настройки системы, интеграции и создание пользователей.*

1.  **Создание пользователей:**
    *   Создайте учетную запись с профилем **«Администратор»** для себя.
    *   Создайте учетные записи для остальных сотрудников, выбрав соответствующие предопределенные профили: `Бухгалтер`, `Кадровик-расчетчик`, `Специалист`, `Менеджер`.
2.  **Настройка Telegram-бота (для подсистемы «Обслуживание клиентов»):**
    *   Выполните первичную настройку Telegram-бота (получите токен и ID группы).
    *   Настройте регламентное задание для автоматической отправки сообщений ботом.
    *   Заполните системные константы полученными данными:
        *   `Токен управления телеграмм ботом`
        *   `Идентификатор группы для оповещения`
3.  **Настройка номенклатуры:**
    *   Создайте (или выберите существующую) номенклатуру с видом «Услуги» (рекомендуемые наименования: *Абонентская плата*, *Работы специалиста*).
    *   Заполните созданными наименованиями константы:
        *   `Номенклатура абонентская плата`
        *   `Номенклатура работы специалиста`
