## Инструкция по запуску автотестирования (Vanessa Automation)

### 1. Подготовка окружения

Для корректного запуска сценариев автотестирования необходимо подготовить чистую базу данных и выполнить ряд предварительных настроек.

#### 1.1. Запуск базы данных
*   Запустите пустую информационную базу с новой конфигурацией.
*   Дождитесь полного выполнения всех регламентных заданий.

#### 1.2. Создание пользователей
Создайте в системе **4 пользователей** с соответствующими правами доступа:

| Роль пользователя | Необходимые права |
| :--- | :--- |
| Менеджер | Профиль `Менеджер по обслуживанию` |
| Специалист | Профиль `Специалист` |
| Бухгалтер | Профиль `Бухгалтер` |
| Кадровик | Профиль `Кадровик-расчетчик` |

#### 1.3. Заполнение констант

Заполните системные константы следующими значениями:

**Данные для тестового Telegram-канала:**
*   **Токен управления телеграмм ботом:** `используте ваш токен`
*   **Идентификатор группы для оповещения:** `используте ваш идентификатор`

**Данные для номенклатуры:**
*   **Номенклатура «Абонентская плата»:** (выберите или создайте элемент справочника)
*   **Номенклатура «Работы специалиста»:** (выберите или создайте элемент справочника)

#### 1.4. Ссылка для вступления в Telegram-канал

*   **Канал:** [Ссылка](https://t.me/+8cKJusY25CRjZWJi)

### 2. Завершение подготовки

После выполнения всех вышеуказанных шагов:
1.  Убедитесь, что все константы заполнены верно.
2.  **Закройте базу данных.** Она готова к прогону автотестов.
