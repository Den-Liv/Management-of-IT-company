
## Инструкция для менеджера
*Работа с заявками клиентов.*

1.  **Прием и регистрация заявки:**
    *   При звонке клиента создайте документ **«Обслуживание клиентов»**.
    *   Зафиксируйте проблему, для решения которой требуется специалист.
2.  **Планирование:**
    *   Найдите свободное время в расписании специалистов.
    *   Запланируйте заявку на выбранное время. *Система автоматически отправит уведомление специалистам через Telegram-бота.*
